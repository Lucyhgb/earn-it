# 赚了么：学生党预算计算器

一款面向学生群体的移动端优先单页应用，帮助用户计算实习、读研期间的日均收支、启动成本回本时间，提供清晰的财务规划建议。

## 核心功能

### 收入管理
- **实习收入**：支持日薪/月薪/时薪三种计薪方式，可设置每周到岗天数
- **读研收入**：国家助学金、课题组补贴、助教/助研津贴、导师额外补贴、横向项目、奖学金
- **家里支持**：周/月/学期多种频率，金额自动日均计算
- **兼职收入**：支持多个兼职项目，灵活配置频率

### 支出计算
- **餐饮支出**：早/午/晚餐多价位设置，支持详细模式和囤货均摊模式
- **通勤支出**：支持步行、单车、电动车、公交/地铁、打车等多种方式
- **住房固定**：房租、水电燃气、网络、管理费，季节性空调/取暖费
- **消耗品**：预设品类快速添加（通用洗护、化妆品、日常用品等）
- **会员订阅**：视频、音乐、网盘/办公等常见订阅服务
- **娱乐医疗**：娱乐支出和医疗健康月预算

### 特色功能
- **启动成本**：一次性成本（去程交通、行李邮寄、押金、安家采购）
- **周期性往返**：学校↔家频率自定义，费用自动日均
- **回本计算**：启动成本 ÷ 日均盈余，计算回本天数
- **弹性储蓄**：设置储蓄目标，规划月均储蓄金额

### 场景预设
- **异地实习**：高薪高消费参数，适合跨省暑期实习
- **本地实习**：低支出模式，适合住校通勤

## 技术特性

- **单文件部署**：所有代码（HTML/CSS/JS）集中在一个HTML文件中
- **移动端优先**：针对320px-420px屏幕优化，支持响应式布局
- **数据持久化**：基于localStorage，自动保存用户输入
- **实时计算**：所有输入变化即时计算，无需提交
- **无需后端**：纯前端应用，可直接部署到任意静态托管服务

## 使用说明

### 快速开始
1. 直接在浏览器中打开 `indexV05.html` 文件
2. 选择身份标签（实习/在校/家里支持/兼职）
3. 使用场景预设快速填充数据，或手动填写
4. 查看底部实时计算的日均盈余和月预估结余

### 数据管理
- 数据自动保存到浏览器localStorage
- 点击"重置所有数据"按钮可清除所有输入
- 切换预设会覆盖当前数据

## 部署指南

### 方式一：直接部署静态文件
将 `indexV05.html` 上传至任意静态托管服务：

- **GitHub Pages**：将文件放入 `gh-pages` 分支
- **Netlify**：拖拽上传HTML文件
- **Vercel**：创建一个新项目并上传
- **阿里云OSS/腾讯云COS**：上传为静态资源
- **任意Web服务器**：Nginx/Apache/Caddy等直接托管

### 方式二：绑定自定义域名
1. 将 `indexV05.html` 重命名为 `index.html`（或配置服务默认读取）
2. 在托管平台绑定您的域名
3. 配置DNS解析指向托管服务

### 方式三：嵌入现有网站
```html
<iframe src="indexV05.html" width="100%" height="100%" frameborder="0"></iframe>
```
